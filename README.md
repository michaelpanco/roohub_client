# Roohub Test Project

## Development Environment Setup

 - nodeJS v14
 - react 17.0.2

 ## Installation

 - Clone the Project
 - git clone `git@github.com:michaelpanco/roohub_client.git`
 - change the `api_base_url` value from `src/config.js` to the url of the roohub_api running in your local machine
 - `run npm install`
 - `run npm start`
